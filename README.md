
## Features

- Displays a list of posts from API Post JSON Placeholder
- Allow users to add new posts
- Use Redux for state management

## Project Structure

- `src/components`
- `src/redux`
- `App.ts`
- `main.ts`
- `src/constant`
- `src/api`

## Tech 

- React
- Redux
- React-redux
- Redux-toolkit
- Axios
- JSON API

## Usage

- App will display a list of posts fetched from API JSON.
- Create form to be added post.
